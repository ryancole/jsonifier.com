API Documentation

The jsonifier API exposes a single resource that allows developers to create a JSON paste over HTTP.

To create a new JSON paste, simply send a POST request containing your JSON data, in the 'json' named parameter, to http://jsonifier.com{{ url_for('api.create') }}. The response will be JSON if successful, otherwise it will be an HTTP error code. A successful paste's JSON response will contain a single key, named 'id', which will be the newly created paste's id.

Example

curl -d "json=%7B%22title%22%3A%22this%20is%20an%20example%20paste%22%7D" http://jsonifier.com/api/create
